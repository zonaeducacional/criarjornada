# A Forja do Limiar

Um jogo educativo interativo sobre a **Jornada do Herói**, desenvolvido para estudantes e entusiastas de literatura e narrativa.

## 🚀 Objetivo do Projeto
O objetivo é ensinar as 13 etapas da Jornada do Herói de Joseph Campbell através de um desafio de ordenação lógica. O usuário deve organizar fragmentos de uma história original ("Arion e o Cristal do Limiar") em suas respectivas etapas narrativas.

## 🛠 Tech Stack
- **Frontend**: HTML5, CSS3 (Glassmorphism & Custom Properties), JavaScript (Vanilla).
- **Ícones**: Lucide Icons via CDN.
- **Efeitos**: Canvas Confetti.
- **PWA**: Suporte offline completo via Service Worker e Web App Manifest.
- **Tipografia**: Cormorant Garamond (Serif) e Plus Jakarta Sans (Sans).

## 📱 Funcionalidades
- **Interface Premium**: Design escuro com detalhes em dourado e efeitos de vidro.
- **Drag & Drop**: Suporte completo para Mouse e Touch (dispositivos móveis).
- **Feedback em Tempo Real**: Dicas automáticas para trechos colocados incorretamente.
- **Instalável**: Transformado em PWA para uso offline como um aplicativo nativo.
- **Celebração**: Animação de confetes ao completar a jornada com sucesso.

## 📦 Como rodar
Basta abrir o arquivo `index.html` em qualquer navegador moderno. Para funcionalidades de PWA (instalação), é necessário servir os arquivos via HTTPS (ou localhost).
